# Thứ tự liên kết
1. data.js
    - Lưu trữ dữ liệu hiển thị
2. setup.js
    - Những function hỗ trợ tạo dialog và event
3. render.js
    - Những function hỗ trợ hiển thị thành phần popup
4. popup.js
    - File js chính, tạo dialog, event và thêm vào DOM
5. openDialog.js
    - Những function hỗ trợ cho event mở dialog
